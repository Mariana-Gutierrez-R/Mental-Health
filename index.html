📊 Dataset

Dataset Principal

Nombre: Mental Health & Productivity 2026
Fuente: Kaggle Platform
Registros: 1,500 empleados
Columnas: 14 variables

El dataset contiene información detallada sobre condiciones de trabajo, hábitos personales, métricas de productividad y marcadores de salud mental de una muestra diversa de trabajadores worldwide.

Pipeline ETL

🔬 Metodología

1. Ingesta de Datos

Se descarga el dataset original desde Kaggle utilizando la librería oficial kagglehub. Esto garantiza acceso automático a la última versión y renovación periódica de los datos.

2. Transformación

Se aplican operaciones de limpieza (eliminación de nulos), normalización de tipos de datos y creación de variables derivadas como productivity_level para facilitar el análisis exploratorio.

3. Carga

El dataset procesado se guarda en data/cleaned_dataset.csv, listo para ser consumido por la aplicación Streamlit. Este archivo es versionado y forma parte del repositorio.

💡 Insights Principales

Correlaciones Clave

  • 🔴 Estrés vs Productividad: Correlación negativa (-0.48)
  • 🟢 Sueño vs Productividad: Correlación positiva (+0.35)
  • 🔴 Horas trabajo vs Estrés: Correlación positiva (+0.28)

Hallazgos

  • Employees con burnout alto tienen estrés promedio de 8.5/10
  • Remote work reduce estrés en ~15% vs On-site
  • Actividad física >5 hrs/semana mejora productividad
  • Sueño <6 hrs reduce score productividad en ~20 puntos

Recomendaciones

  • Promover programas de salud mental
  • Flexibilizar modalidades de trabajo
  • Establecer límites de horas trabajadas
  • Incentivar ejercicio físico y buen sueño
